# Monitor web Comisaría Virtual

_Monitor web para el sitio de comisariavirtual.cl_

## Instalación

```bash
npm install --save-dev
npx eleventy
```

Correr con:

```bash
npx eleventy --serve
```

### Usa
* [11ty](https://www.11ty.dev/)
* [UptimeRobot](https://uptimerobot.com/)
* [Netlify](https://www.netlify.com/)
