Acerca de: Monitor Web

¿Qué es esto?

¿Qué es esto?

Es un monitor web para el sitio online de la comisaría virtual . Al iniciar el monitor, avisará cada 5 minutos si ese sitio web está caído o no disponible.
¿Cómo funciona?

¿Cómo funciona?

El sitio web de la comisaría virtual es monitoreado utilizando UptimeRobot, la cual monitorea cada 5 minutos (versión gratuita). Esta app se ejecuta por parte del cliente (navegador) y usa la API de UptimeRobot, enviando una clave pública, para recibir y notificar el estado del sitio web.
¿Por qué?

¿Por qué?

Principalmente porque quería hacer un proyecto que pudiera ser útil (en tiempos de pandemia), mientras aprendo Eleventy y Netlify.
¿Qué tecnología usa?

¿Qué tecnología usa?

Hecho “a mano” con HTML5, CSS3 y JavaScript, junto con Eleventy , Netlify y UptimeRobot.
